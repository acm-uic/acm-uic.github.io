# Servers Overview

ACM main infrastructure servers are currently racked at CDRLC 2422. Currently, we have 8 servers available, 5 of which are in production which are

- [Avocado](https://acm-uic.github.io/docs/infrastructure/servers/avocado)
- [Bharat](https://acm-uic.github.io/docs/infrastructure/servers/bharat)
- [Chase](https://acm-uic.github.io/docs/infrastructure/servers/chase)
- [Lee](https://acm-uic.github.io/docs/infrastructure/servers/lee)
- [Miku](https://acm-uic.github.io/docs/infrastructure/servers/miku)

The 3 servers that are in the process of deploying to production are

- [Mikrotik Rose](https://mikrotik.com/product/rds2216)
- Roxy (Dell Poweredge)
- Eris (Dell Poweredge)

The networking for the whole rack (and more) is managed by 2 pfSense routers in:

- Ackem
- Littlebell
